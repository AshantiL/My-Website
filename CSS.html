Learning about CSS

  • CSS contains style rules which informs a browser how to represent a document. There are several ways to link these rules of style to HTML documents , but the easiest method for beginning is to use the STYLE element of HTML. It is basically placed into the HEAD tag and it encloses the style rules of that page.
  • CSS is perfect to permit the separation of structure. CSS contains all the elements of HTML such as font colors and its styles, background mode, alignments of elements and much more. All these are stored in a class and when you want to use that style then you need to call for that class.
  • CSS shares some drawback with programming languages. Especially, the problem of selecting the suitable names of classes for CSS. To select descriptive names for class, developers might connect the class with presentation features.
